⚡ React + Vite Setup
✅ 1. Create Project

Open terminal:

npm create vite@latest

✅ 2. Project Details

It will ask:

✔ Project name: my-app
✔ Select a framework: React
✔ Select a variant: JavaScript (or TypeScript)

✅ 3. Go into project
cd my-app

✅ 4. Install dependencies
npm install

✅ 5. Run project
npm run dev


✅ Install React Router
npm install react-router-dom

✅ Install Axios
npm install axios

✅ Install Socket.IO Client (for your project 👀)
npm install socket.io-client
